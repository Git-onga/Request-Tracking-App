# RequestBoard

- Live demo: [https://request-tracker-e6f992.netlify.app](https://request-tracker-e6f992.netlify.app)

RequestBoard is a premium, modern dashboard designed to capture, organize, and track various types of user inquiries and requests. Featuring high-performance filtering, real-time search, and dynamic visual aesthetics, it serves as a streamlined portal for collecting user feedback and tracking its resolution.

## Features

- **Submit Inquiries**: A structured submission form collects user details (Name and Email), Request Type, Priority level, and a descriptive message.
- **Request Categorization**: Organizes items under distinct categories: *Bug*, *Feature Request*, *General Feedback*, *Partnership*, and *Other*.
- **Priority Scaling**: Labels entries as *Low*, *Medium*, or *High* priority to assist in triage and sorting workflows.
- **Lifecycle Status Tracking**: Progress requests through their lifecycle with status updates: *New*, *In Review*, *Resolved*, and *Rejected*.
- **Interactive Dashboard**:
  - **Dynamic Statistics**: Real-time counter chips display aggregates of requests across all status levels.
  - **Advanced Filtering**: Instantly narrow down requests by request type or status.
  - **Real-Time Search**: Search submissions instantly by user name, email address, or message details.
  - **Sorting**: Order entries by newest first, oldest first, or priority level.
- **Detailed Insights Modal**: Select any card to open a detailed dialog card to inspect full message details, update the ticket status, or delete the request.
- **Dual-Theme Mode**: Toggle between a dark aesthetic and a light layout with automatic theme memory.
- **Data Persistence**: Retains all submissions locally on page refresh.
